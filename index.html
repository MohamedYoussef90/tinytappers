Help & Settings

This interactive website is designed for young children (1+ years) to explore keyboard and touch interactions.

How to use:

  • Press any key on the keyboard to see animations and hear sounds
  • Touch/click the screen on mobile devices
  • Each letter key shows an animal
  • Each number key shows a number emoji
